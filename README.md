# Northwind AI Chat Interface 

# 🧠 Keydraft Chat Insights

An AI-powered chat interface that connects to the classic **Northwind SQLite** database and allows users to:
- Ask questions in natural language
- Receive smart SQL queries generated by OpenAI or Gemini
- View summarized answers
- Automatically render charts (Bar, Line, Pie) based on the query result

## 🚀 Features

- ✅ Ask any business analytics question about Northwind database
- ✅ Backend auto-generates SQL using Gemini or OpenAI
- ✅ Executes on SQLite and returns structured data
- ✅ Frontend renders data as:
  - Bar Chart
  - Pie Chart
  - Line Chart
- ✅ Clean and responsive UI
